## How I think about building

- Shipped > perfect. Real users > hypothetical TAM.
- If I can self-host it, I self-host it. Understand your stack end-to-end.
- Brutally honest evaluation over optimistic framing — every idea gets stress-tested.
- Targeting developed markets: build once, price honestly, don't fight willingness-to-pay problems.
